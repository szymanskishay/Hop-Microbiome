NOTE: 
These are the outputs from bioinformatic analysis as described in the associated manuscript.
Importantly, this means the "constax_taxonomy" files are not necessarily  the updated versions with taxonomy identified by NCBI blast. 
ITS refers to the fungal portion, while 16S refers to the bacterial.

The metadata in the "hopmap" files should be sure to use the files specific for their respective marker (ITS vs 16S).

The ITS folder also contains the xlsx file used from Funguild.
