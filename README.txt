These are codes used in the publication of the manuscript titled "Temporal changes in fungal and bacterial communities in developing hop cones and rhizomes over a single growing season".

Codes for specific figures and analyses have been drawn out in the main directory, with subsets detailed towards each marker (ITS vs 16S) are contained in their sub-directories. Notably, code that generated the data that has been presented as Tables herein are in the following directories:
Table 1: Assembled from the B-dispersion scripts in both ITS and 16S directories
Table S1: Assembled from the Core-Microbiome-ITS.R script in the ITS directory
Table S2: Assembled from the core_shade-16S.R script in the 16S directory.

For clarities sake, we note that code for Figure 6 and supplementary figures 1, 2 and 5 are in Figures-6-S1-S2-S5.R The are grouped by their commonality of being relative abundance plots

The OTU tables, sequences, taxonomic assignments, and metadata are also provided in the /data directory. 

Due to potential changes in dependencies and packages between the time of writing the code, publication, and the indefinite future where this repository may be being accessed, we note that it is possible that some codes may not work as initially intended or have issues with installation, and apologize for any inconvenience this provides. We note that the scripts utilizing the package MicrobiomeAnalystR can experience installation difficulties or conflicts with other installations.

Raw sequence information has been deposited on NCBI as Bioproject PRJNA1295563. 

Codes may be adapted from other studies. In particular, the core microbiome codes are adapted from the following paper:
Shade, A. and Stopnisek, N. 2019. Abundance-occupancy distributions to prioritize plant core microbiome membership. Curr. Opin. Microbiol. 49:50-58.

Generative AI and AI-based coding agents were not used at any point in the writing of this code.
